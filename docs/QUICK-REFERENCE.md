3. Work on task
4. Update [PROGRESS-TRACKER.md](./PROGRESS-TRACKER.md)
5. Mark task as @completed

### 🐛 **Fix Bug**
```bash

1. Check [BUGS-ISSUES.md](./BUGS-ISSUES.md) for details
2. Create task in [TASKS.md](./TASKS.md) if needed
3. Fix the bug
4. Mark issue as @completed

```
